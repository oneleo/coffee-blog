## 特色
- 乾香：莓果香、果汁的酸甜味、淡淡油脂香氣
- 濕香：草莓甜味、熟果香、甜甜香料味
- 啜吸：它的甘甜度和莓果香氣絕對會讓您驚豔不已！
甜度、稠度、厚度相當良好，有著類似香蕉乾的熟果甜味。

- 沖泡的時候濃濃的咖啡香散落在客廰的各個角落，好似美好的愛情即將要有了結果
